Certification Mikrotik MTCNA

MIkrotik Certified Network Associate (MTCNA) This training will provide you with the skills to configure a MikroTik RouterOS Routerboard as a dedicated router, a bandwidth manager, a firewall, a wireless access point, a VPN Server etc. This course covers the basics skills of routerOS configuration which is very useful for those that are new to Mikrotik RouterOS.

Certification Mikrotik MTCRE

MIkrotik Certified Routing Engineer (MTCRE) This advanced training will be focusing on building a routed network using Mikrotik equipment. With routing, you could make your network to be scalable, manageable, and secure, eliminating unnecessary NAT on internal network. Topics that will be covered in this are static routing, OSPF, load balancing, VPN, and VLAN.

Certification Mikrotik MTCTCE

MIkrotik Certified Traffic Control Engineer (MTCTCE) This advanced training will provide you with the skills to configure a MikroTik RouterOS as a bandwidth manager for your network, Identify network application with firewall mangle, setup a secured DNS cache/proxy, setup DHCP relay, and deploy a mikrotik-based proxy server.
